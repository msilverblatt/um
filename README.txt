Universal Machine Emulator
Michael Silverblatt
Spring '13

Built in C for Tufts Comp 40 - Computer Architecure.
Reads and runs universal binary programs in a specified format. 
Emulates 32-bit hardware with 8 registers which can handle 14 opcode instructions.
Profiled and optimized using kcachegrind and the DDD debugger.

Requires course-specific software to compile.
